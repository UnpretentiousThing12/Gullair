Gull Simulator - Third Person Prototype

Contents:
- index.html: Main game file. Runs a 3D scene using Three.js with:
  * Flat sea, blocky cliffs, green cube gull
  * Third-person camera follow
  * WASD/Arrow key movement
  * Responsive to window resize

To run:
Open index.html in a modern browser (Chrome, Firefox, Edge). No install needed.
